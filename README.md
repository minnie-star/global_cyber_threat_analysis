# Global Cyber Threat Intelligence Analysis

# Overview

This project explores global cyber threat intelligence data to strengthen my skills as a software engineer in data analysis, visualization, and security insights. The dataset contains records of suspicious and malicious IP addresses, enriched with abuse confidence scores, geographic metadata, and temporal features.  

Dataset source: [Global Cyber Threat Intelligence Dataset 2026](https://example-dataset-link.com)  

The purpose of this software is to practice analyzing real-world cybersecurity data, answer meaningful questions about attack patterns, and build reusable code for future security-focused projects.  

[Software Demo Video](https://youtu.be/SPaLo4eFKpU)

# Data Analysis Results

**Questions and Answers:**
- **Which countries have the highest number of suspicious IPs?**  
  United States leads with over 4,000 flagged IPs, followed by China, Germany, and several European countries.  

- **What is the average abuse confidence score per country?**  
  All entries have a score of 100, meaning every IP in the dataset is fully malicious.  

- **What monthly trends are visible?**  
  All 10,000 records fall into April 2026, showing dataset limitations but also highlighting a concentrated attack period.  

- **Which IP addresses are most frequently reported?**  
  Several repeat offenders appear multiple times, indicating persistent malicious activity.  

- **How are attacks distributed by continent, time of day, and weekday?**  
  North America dominates continent counts, evenings show the highest attack frequency, and weekday analysis reveals consistent activity across the week.
