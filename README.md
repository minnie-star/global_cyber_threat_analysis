# Global Cyber Threat Intelligence Analysis

## Overview
This project analyzes global cyber threat intelligence data to identify:
- Countries with the highest number of suspicious IP addresses
- Average abuse confidence score of malicious IPs per country
- Monthly spikes in malicious activity

The analysis is implemented in **Python** using **Pandas** for data manipulation and **Matplotlib** for visualization.
